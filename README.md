# World Cup Data Warehouse

## 📌 Project Overview
This project designs and implements a star schema data warehouse for analyzing FIFA World Cup team performance.

## 🧠 Data Model
- Fact table: Fact_Team_Tournament_Performance
- Dimensions:
  - Dim_Team
  - Dim_Tournament
  - Dim_Group
  - Dim_Coach

## 🏗️ Tools
- SQL Server (SSMS)
- Git & GitHub
- draw.io (ERD design)
